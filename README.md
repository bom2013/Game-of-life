# Game-of-life
Python implementation of Game of life
## How to use
* Download [GameOfLife.py](../blob/master/GameOfLife.py)
* run it :)  

**Note**: The program start in pause mode
## Keyboard shortcuts
* Space -- Pause\Start
* R -- Reset
* -\\+ -- Decrease\\Increase speed
